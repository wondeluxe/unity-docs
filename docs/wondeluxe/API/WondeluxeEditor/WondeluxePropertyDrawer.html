WondeluxePropertyDrawer Class

declarationpublic class WondeluxePropertyDrawer : PropertyDrawer;
inheritsPropertyDrawer
namespaceWondeluxeEditor
assemblyWondeluxeEditor

Base class for creating custom PropertyDrawers for use in thecustom inspector solution. Supports multiple PropertyDrawers used on the same member as long as only one PropertyDrawer may modify the property's label, and only one PropertyDrawer may modify the property's layout (way GUI is drawn). Can be mixed with Unity's default PropertyDrawers as long as no other PropertyDrawers modify the property's layout.

Public properties

bool HasAdditionalLayout { get; }

Override to return trueif the property drawer appends content to the property's appearance. If true, the property drawer must override GetAdditionalPropertyHeightand OnAdditionalGUI.
bool HasCustomLabel { get; }

Override to return trueif the property drawer changes the property's label. If true, the property drawer must override GetCustomLabel.
bool HasCustomLayout { get; }

Override to return trueif the property drawer changes the appearance of the property. If true, the property drawer must override OnCustomGUIand optionally GetCustomPropertyHeight.
bool ValidateObjectSelection { get; }

Override to return trueif the property drawer is a validator for an object property. If true, the property drawer must override GetValidObjectReference.

Override to provide a modified height for the property. The returned height should not factor in additional height for appended content. Will only be called if HasCustomLayoutis overridden to return true.
