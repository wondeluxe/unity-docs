PrefabPool Class

declarationpublic class PrefabPool : MonoBehaviour;
inheritsMonoBehaviour
namespaceWondeluxe
assemblyWondeluxe

Component for managing pre-initialized instances of a prefab.

Public properties

int AvailableCount { get; }

The number of available (not requested) prefab instances remaining in the PrefabPool.
bool Expand { get; set; }

If true, additional instances of Prefabwill be instantiated if the allocation is exhausted when Requestis called.
PrefabPoolFillMode FillMode { get; set; }

Mode defining when prefab instances are created.
Transform Parent { get; set; }

The transform available/inactive (not requested) instances of the prefab will be parented to. If null, instances will be parented to the PrefabPool's transform.
GameObject Prefab { get; set; }

The prefab to create instances of.
int Size { get; set; }

Total capacity of the PrefabPool. Indicates the number of instances created by the pool.

Public methods

void Clear ()

Empties the PrefabPool and destroys all available (not requested) instances.
void Fill ()

Creates the number of prefab instances defined by the PrefabPool's Size.
void Refill ()

Used to create or destroy prefab instances after the PrefabPool's Sizehas been modified. Instantiates additional instances of Prefabif Size has been increased, destroys available (not requested) instances if Size has been decreased.
void ReparentAvailableInstances ()

Reparents available (not requested) prefab instances to the PrefabPool's Parenttransform. This method does nothing if Parent is null.
GameObject Request (bool active)

Gets a prefab instance from the PrefabPool if one is available, or if Expandis true.
bool Request (bool active, out GameObject instance)

Gets a prefab instance from the PrefabPool if one is available, or if Expandis true.
T Request <T> (bool active)

Gets a prefab instance from the PrefabPool if one is available, or if Expandis true, and returns one of its components.
bool Request <T> (bool active, out T component)

Gets a prefab instance from the PrefabPool if one is available, or if Expandis true, and returns one of its components.
bool Return (GameObject instance)

Returns a prefab instance to the PrefabPool, making it available for future requests.
bool Return (Component component)

Returns the GameObject of a Component to the PrefabPool, making it available for future requests.
