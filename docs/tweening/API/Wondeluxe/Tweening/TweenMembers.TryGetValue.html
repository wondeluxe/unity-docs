TweenMembers.TryGetValue Method

declarationpublic bool TryGetValue <T> (string name, out T value);
namespaceWondeluxe.Tweening
assemblyWondeluxe.Tweening

Attempts to get the value a member will be tweened to.

Type parameters

T

The type of the value to be tweened.

Parameters

string name

The name of the member to tween.
out T value

When this method returns, contains the value the member with the specified name will be tweened to; if no member with the specified name is found, the default value for the typewill be used.

Returns

bool

trueif a member with the specified name is contained in the set, otherwise false.
