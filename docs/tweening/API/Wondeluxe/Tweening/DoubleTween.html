declarationpublic struct DoubleTween;
namespaceWondeluxe.Tweening
assemblyWondeluxe.Tweening

Lightweight structure for handling a tween between two double values.

DoubleTween provides efficient logic for tweening between two double values at the expense of features provided by Tween. Clients are expected to manage the update cycle of DoubleTween and apply updates to target objects where required.
