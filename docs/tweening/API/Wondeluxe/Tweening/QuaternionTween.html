QuaternionTween Struct

declarationpublic struct QuaternionTween;
namespaceWondeluxe.Tweening
assemblyWondeluxe.Tweening

Lightweight structure for handling a tween between two Quaternion values.

QuaternionTween provides efficient logic for tweening between two Quaternion values at the expense of features provided by Tween. Clients are expected to manage the update cycle of QuaternionTween and apply updates to target objects where required.
