Vector2IntTween Struct

declarationpublic struct Vector2IntTween;
namespaceWondeluxe.Tweening
assemblyWondeluxe.Tweening

Lightweight structure for handling a tween between two Vector2Int values.

Vector2IntTween provides efficient logic for tweening between two Vector2Int values at the expense of features provided by Tween. Clients are expected to manage the update cycle of Vector2IntTween and apply updates to target objects where required.
