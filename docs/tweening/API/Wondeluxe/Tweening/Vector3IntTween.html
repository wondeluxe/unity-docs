Vector3IntTween Struct

declarationpublic struct Vector3IntTween;
namespaceWondeluxe.Tweening
assemblyWondeluxe.Tweening

Lightweight structure for handling a tween between two Vector3Int values.

Vector3IntTween provides efficient logic for tweening between two Vector3Int values at the expense of features provided by Tween. Clients are expected to manage the update cycle of Vector3IntTween and apply updates to target objects where required.
